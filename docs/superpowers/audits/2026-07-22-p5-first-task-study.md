# P5 · 首任务用户测试协议（禁止现场教学）

## 目标

5–8 名目标用户；每人独立完成「登录 → 首页 → 首任务 → 看到成果」。主持人只观察，不提示操作路径。

## 记录字段

| 用户 | 完成 | TTFV(s) | 犹豫点 | 错误恢复 | 品质 1–5 |
|------|------|---------|--------|----------|----------|
| U1 | | | | | |
| U2 | | | | | |

- **完成**：是否到达可检查成果（文件/摘要/可追问）
- **TTFV**：从进入首页到确认首个有价值成果的秒数
- **犹豫点**：停顿 >5s 的界面元素
- **错误恢复**：缺模型 / 网络 / 模型失败是否自行恢复

## 发布门槛

- 首任务完成率相对基线上升
- TTFV 明显下降
- 路由无整屏闪烁
- 核心动画稳定约 60fps
- 键盘可达 + `prefers-reduced-motion` 可用

漏斗事件：`auth_completed` / `first_task_started` / `first_value_confirmed` / `d1_retained` / `d7_retained`（本地 cohort A/B）。
